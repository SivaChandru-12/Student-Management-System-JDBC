# Student Management System (JDBC)

## Overview

This project is a console-based Student Management System developed using Java JDBC and Oracle Database. It demonstrates database connectivity, CRUD operations, and layered architecture.

## Features

- Add Student
- Find Student by ID
- Update Student Marks
- Oracle Database Connectivity
- Exception Handling
- Layered Architecture

## Technologies Used

- Java
- JDBC
- Oracle Database 21c
- Eclipse IDE

## Project Structure

- Core Java
- JDBC
- PreparedStatement
- ResultSet
- Exception Handling
- OOP
- Packages

STUDENT
